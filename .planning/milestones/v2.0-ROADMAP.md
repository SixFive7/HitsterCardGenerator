## Overview

Transform the console application into a modern web interface using .NET Minimal API backend with Svelte + Tailwind frontend. Users can now upload CSV files via drag-drop, match songs with Spotify through a visual interface, preview and customize cards in an interactive carousel, and export print-ready PDFs.

## Phases

### Phase 5: Web Foundation

**Goal**: Set up Minimal API project, Svelte/Vite frontend with Tailwind, establish project structure
**Depends on**: v1.0 complete
**Plans**: 2 plans

Plans:
- [x] 05-01: Web SDK & Minimal API Foundation
- [x] 05-02: Svelte + Tailwind Frontend

**Details:**
- .NET Minimal API on port 5000 with health endpoints
- Svelte 5 with runes ($state, $effect) for modern reactivity
- Tailwind v4 with @tailwindcss/vite plugin (zero-config)
- Spotify-inspired color scheme (#1DB954 green, #191414 dark)

### Phase 6: File Upload
